## 🔧 Customization Made Easy

### Change Colors
**File**: `app/globals.css`
```css
--primary: #1e3a5f    /* Change primary color */
--accent: #ea580c     /* Change accent color */
```

### Add Products
**File**: `lib/mock-data.ts`
- Add to `mockProducts` array
- Follow Product interface

### Modify Tax Rate
**File**: `app/page.tsx` (line 15)
```typescript
const [taxRate, setTaxRate] = useState<number>(0.15) // 15%
```

### Add Store Location
**File**: `components/sidebar.tsx` (lines 11-16)
- Add to stores array

### Change Configuration
**File**: `config/app.config.ts`
- Modify any setting centrally
